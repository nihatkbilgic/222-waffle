# 222 Waffle Website 🧇

Modern, hızlı ve responsive bir waffle satış web uygulaması. Kullanıcı deneyimi, performans ve mobil uyumluluk odaklı geliştirilmiştir.

---

## 🎯 Proje Amacı
Bu proje, bir waffle markası için dijital menü ve sipariş önizleme arayüzü sunmak amacıyla geliştirilmiştir. Temel hedef; sade, hızlı ve satış odaklı bir kullanıcı deneyimi oluşturmaktır.

---

## ✨ Özellikler
- 📱 Tam responsive tasarım (mobil, tablet, desktop uyumlu)
- 🍓 Dinamik ürün listeleme
- ⚡ Hızlı yüklenen modern UI
- 🧭 Kullanıcı dostu navigasyon
- 🎨 Minimal ve satış odaklı tasarım

---

## 🛠 Kullanılan Teknolojiler
- HTML5
- CSS3 (Flexbox / Grid)
- JavaScript (Vanilla veya framework varsa belirt)
- Vercel / Netlify (deploy)
- AI destekli geliştirme süreci
